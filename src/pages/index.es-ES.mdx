# Introducción

¡Este sitio te permite aprender y usar las transacciones básicas del XRP Ledger!

No es necesario tener conocimientos de un lenguaje de programación específico, como Javascript o Python, para enviar transacciones en un entorno de Test.

¡Cada página incluye además un ejemplo de código para cada lenguaje de programación, así puedes usarlo como referencia cuando escribas código en cada lenguaje!
